

# 1. 전체 흐름 (이번 파트에서 하는 것)

- 딥러닝 시작
- **TensorFlow 2.0으로 인공 신경망 구현**
- 실제 데이터로 **예측 모델 만들기**

👉 목표
→ 고객이 은행을 떠날지(이탈) 예측하는 모델 만들기

------

# 2. 딥러닝 핵심 개념

- 딥러닝 = **인공 뇌를 만드는 것**
- 구조:
  - 뉴런(Neuron)
  - 레이어(Layer)
  - Fully Connected 구조

👉 이번에는

- CNN 같은 건 안 쓰고
- **완전 연결 신경망(ANN)**만 사용

------

# 3. 문제 유형

- **이진 분류 (Binary Classification)**

👉 결과:

- 1 → 고객이 떠남
- 0 → 고객이 유지됨

------

# 4. 데이터셋 설명 (은행 고객 데이터)

각 고객 정보:

- Credit Score (신용 점수)
- Geography (국가)
- Gender (성별)
- Age (나이)
- Tenure (가입 기간)
- Balance (잔액)
- NumOfProducts (사용 상품 수)
- Has Credit Card (카드 보유 여부)
- Is Active Member (활동 여부)
- Estimated Salary (추정 연봉)

👉 결과 변수:

- Exited
  - 1 → 떠남
  - 0 → 유지

------

# 5. 이 문제의 의미 

- **Churn = 고객 이탈**

👉 목적:

- 누가 떠날지 미리 예측
- 떠날 가능성이 높은 고객에게
  → 혜택 제공해서 유지

------

# 6. 전체 구현 단계 (매우 중요)

## Part 1: 데이터 전처리

- 데이터 불러오기
- 필요 없는 컬럼 제거
- 인코딩 (문자 → 숫자)
- 스케일링

------

## Part 2: 신경망 만들기

- 모델 초기화
- 입력층 + 은닉층 1
- 은닉층 2
- 출력층

------

## Part 3: 학습

- 컴파일
  - optimizer
  - loss function
- 학습 진행 (fit)

------

## Part 4: 예측 & 평가

- 새로운 고객 예측
- 테스트 데이터 예측
- Confusion Matrix
- 정확도 확인

------

# 7. 중요한 포인트

- 데이터셋이 이전보다 **현실적 (feature 많음)**
- 단순 템플릿 + 추가 전처리 필요
- 결과는 단순 Yes/No가 아니라
  👉 **확률로도 예측 가능**

------
