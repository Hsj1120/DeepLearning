## ✔ 입력 형식 (중요)

```python

```

👉 반드시 **2차원 배열**


## ✔ 범주형 데이터 처리

```python

```

👉 학습할 때 사용한 **같은 스케일링 적용**

- `fit_transform` ❌
- `transform` ⭕

# 4. 성능 평가 (Confusion Matrix)

```python
from sklearn.metrics import confusion_matrix, accuracy_score 
// confusion_matrix → 예측 결과를 표로 정리
// accuracy_score -> 정확도 계산
cm = confusion_matrix(y_test, y_pred)
// 정확도 계산 y_test -> 실제값, y_pred -> 예측값
print(cm) 
accuracy_score(y_test, y_pred)
```

👉 예측 vs 실제 비교

| 구분           | 의미          |
| -------------- | ------------- |
| True Positive  | 맞게 예측 (1) |
| True Negative  | 맞게 예측 (0) |
| False Positive | 틀림          |
| False Negative | 틀림          |
