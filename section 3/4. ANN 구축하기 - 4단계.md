# ANN 학습 과정 (Compile + Fit)

## 1. 전체 흐름

1. 모델 컴파일 (`compile`)
2. 모델 학습 (`fit`)

👉 구조만 만든 상태 → 이제 학습시키는 단계

------

# 2. 모델 컴파일

```python
ann.compile(optimizer='adam', loss='binary_crossentropy', metrics=['accuracy'])
```

## ✔ optimizer (최적화 방법)

- `'adam'` 사용
- 역할:
  👉 가중치를 업데이트해서 **오차(loss)를 줄임**
- 내부적으로 SGD 기반

------

## ✔ loss (손실 함수)

- `'binary_crossentropy'`

👉 예측값 vs 실제값 차이를 계산

### 규칙

- 이진 분류 → `binary_crossentropy`
- 다중 분류 → `categorical_crossentropy`

------

## ✔ metrics (평가지표)

- `'accuracy'`

👉 모델 성능 확인용 (정답률)

------

# 3. 모델 학습

```python
ann.fit(X_train, y_train, batch_size=32, epochs=100)
```

------

## ✔ X_train, y_train

- X: 입력 데이터
- y: 정답 데이터

------

## ✔ batch_size = 32

👉 한 번에 학습하는 데이터 개수

- 1개씩 → 느림
- 여러 개 묶어서 → 효율적

👉 보통 기본값: 32

------

## ✔ epochs = 100

👉 전체 데이터를 몇 번 반복해서 학습할지

- 1 epoch = 전체 데이터 1번 학습
- 100 epochs = 100번 반복

------

# 4. 학습 과정에서 일어나는 일

1. 예측 수행
2. loss 계산
3. optimizer가 가중치 수정
4. 반복하면서 정확도 증가

------

# 5. 중요한 포인트

👉 학습이 진행될수록

- loss ↓ (감소)
- accuracy ↑ (증가)

👉 일정 시점에서 수렴

------

# 6. 핵심 코드 정리

```python
ann.compile(
    optimizer='adam',
    loss='binary_crossentropy',
    metrics=['accuracy']
)

ann.fit(
    X_train,
    y_train,
    batch_size=32,
    epochs=100
)
```

------

# 7. 한 줄 핵심

👉 **compile = 학습 방식 설정**
👉 **fit = 실제 학습 실행**
