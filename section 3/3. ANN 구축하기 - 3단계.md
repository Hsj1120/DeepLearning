# 인공신경망(ANN) 구축 과정 핵심 정리

## 1. 전체 흐름 (4단계)

1. 모델 생성 (Sequential)
2. 입력층 + 첫 번째 은닉층 추가
3. 두 번째 은닉층 추가
4. 출력층 추가

👉 순차적으로 레이어를 쌓는 구조

------

# 2. 모델 생성

```python
ann = tf.keras.models.Sequential()
```

- tf -> 딥러닝 라이브러리
- keras -> TensorFlow안에 포함된 고수준 API
- models -> 모델 관련 기능 모음
- Sequential -> 레이어를 순서대로 쌓는 모델 클래스
- 신경망을 **레이어들의 순서(sequence)**로 구성
- 하나의 객체로 모델 생성

------

# 3. 은닉층 추가 (Dense 사용)

```python
ann.add(tf.keras.layers.Dense(units=6, activation='relu'))
```

## ✔ Dense

- 완전연결층 -> 이전 층의 모든 뉴런이 현재 층의 모든 뉴런과 연결된다

## ✔ units

- 뉴런 개수 (여기서는 6)

## ✔ activation='relu'

- 은닉층에서 사용하는 활성화 함수
- 비선형성 추가

------

## ✔ 은닉층 추가 방법

```python
ann.add(...)
ann.add(...)
```

👉 그냥 **복붙해서 계속 추가하면 됨**

- 첫 번째 은닉층
- 두 번째 은닉층

------

## ✔ 뉴런 개수 결정

👉 정답 없음
👉 **실험(하이퍼파라미터 튜닝)**으로 결정

------

# 4. 출력층 추가

```python
ann.add(tf.keras.layers.Dense(units=1, activation='sigmoid'))
```

## ✔ units = 1

- 이진 분류 (0 or 1)

## ✔ activation='sigmoid'

- 출력값을 확률(0~1)로 변환

------

## ✔ 다중 분류일 경우

- 클래스 3개 → units = 3
- one-hot encoding 사용

------

# 5. 활성화 함수 정리

| 위치          | 함수    |
| ------------- | ------- |
| 은닉층        | ReLU    |
| 출력층 (이진) | Sigmoid |

------

# 6. 핵심 구조 한 번에 보기

```python
ann = Sequential()

ann.add(Dense(units=6, activation='relu'))
ann.add(Dense(units=6, activation='relu'))

ann.add(Dense(units=1, activation='sigmoid'))
```

------

# 7. 핵심 한 줄 정리

👉 **신경망은 Sequential로 만들고, Dense로 레이어를 쌓는다**
