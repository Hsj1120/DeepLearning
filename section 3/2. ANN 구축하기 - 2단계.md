```python
X[:, 2] = labelencoder.fit_transform(X[:, 2])
```

## (2) Geography → One-Hot Encoding

```python
ColumnTransformer + OneHotEncoder
```

```python
ct = ColumnTransformer(
    transformers=[('encoder', OneHotEncoder(), [1])],
    /*
    encoder : 아무 이름 가능
    OneHotEncoder() : 방법
    [1] : 1번 열
    */
    remainder='passthrough'
X = np.array(ct.fit_transform(X)) 
// numpy형태로 변환해서 사용 , fit -> 어떤 값이 있는지 학습, transform -> 실제 변환
)
```

결과:

- France → [1,0,0]
- Spain → [0,1,0]
- Germany → [0,0,1]
- 3개의 열로 늘어난다.

# 6. Feature Scaling (매우 중요 ⭐)

👉 딥러닝에서는 필수

```python
StandardScaler
```

특징:

- 모든 feature 스케일링 (0/1도 포함)
- 데이터를 평균 0, 표준편차 1로 맞춰서 학습이 잘 되게 한다.
- xscaled = x−평균 / 표준 편차
- 예

| feature | 값 범위    |
| ------- | ---------- |
| Age     | 20 ~ 60    |
| Balance | 0 ~ 100000 |

- Balance가 너무 커서 모델이 그것만 중요하게 보기 때문에 모든 feature를 비슷한 크기로 맞춤

```python
from sklearn.preprocessing 
import StandardScaler 
sc = StandardScaler() // 스케일링 도구 만들기
X_train = sc.fit_transform(X_train) // fit -> 평균, 표준편차 계산, transform -> 실제 변환
X_test = sc.transform(X_test) 
// test는 fit하면 안된다. fit은 평균과 표준편차를 계산하는데, 데이터의 정보를 학습하는 과정 즉 미리 보는 것
```

이유:

- 학습 속도 향상
- 안정적인 수렴

# 7. 중요한 개념 정리

### 왜 Feature Scaling 필수?

- 신경망은 **가중치 업데이트** 기반
- 값 크기 차이 크면:
  → 학습 느려짐 / 불안정

------

### 왜 train만 fit?

```python
sc.fit(X_train)
sc.transform(X_test)
```

👉 이유:

- **데이터 누수 방지 (Data Leakage)**
