# 1. 이번 섹션에서 배우는 것

이번부터는
**Self-Organizing Maps**, 줄여서 **SOM**을 배운다.

한국어로는:

**자기조직화 지도**

라고 한다.

------

# 2. SOM을 먼저 이해해야 하는 이유

먼저 배울 내용은:

**SOM이 어떻게 동작하는가?**

이다.

👉 이유:

SOM이 최종적으로 무엇을 하려는지 알아야 나중에 학습 과정을 봤을 때 이해가 된다.

즉:

- SOM의 목표가 무엇인지 이해
- 그 목표를 향해 어떻게 학습하는지 이해
- 결과를 어떻게 해석해야 하는지 이해

이 순서로 가야 한다.

------

# 3. SOM의 핵심 목표

SOM은 복잡한 데이터를
**2차원 지도처럼 표현하는 알고리즘**이다.

쉽게 말하면:

👉 비슷한 데이터끼리는 가깝게 놓고
👉 다른 데이터끼리는 멀리 놓는다.

예를 들면:

- 비슷한 고객끼리 가까이 배치
- 비슷한 이미지끼리 가까이 배치
- 비슷한 패턴을 가진 데이터끼리 가까이 배치

이런 식이다.

------

# 4. K-Means Clustering을 복습하는 이유

그다음에는
**K-Means Clustering**을 빠르게 복습한다.

K-Means는:

**비슷한 데이터끼리 그룹으로 묶는 알고리즘**

이다.

👉 SOM도 결국
데이터의 유사성을 이용하기 때문에
K-Means를 먼저 보면 이해가 쉬워진다.
