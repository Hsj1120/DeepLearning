# Boltzmann Machine 섹션 개요

## 1. 이번 섹션에서 배울 내용

이번 섹션에서는 **Boltzmann Machine**에 대해 배운다.

여기서 다루는 것은 단순히 RBM만이 아니라,
처음 설계된 **원래 형태의 Boltzmann Machine**이다.

즉, **Boltzmann Machine 구조 → 에너지 기반 모델 → RBM → Contrastive Divergence → DBN/DBM**

순서로 학습한다.

------

## 2. Boltzmann Machine이란?

Boltzmann Machine은 확률 기반의 신경망 모델이다.

일반적인 신경망처럼 입력을 넣고 바로 출력을 계산하는 방식이라기보다, 

네트워크 안의 뉴런들이 서로 연결되어 있고 그 상태를 바꿔가며 더 안정적인 상태를 찾아가는 구조이다.

여기서 중요한 개념은 **에너지(Energy)**이다.

Boltzmann Machine은 에너지가 낮은 상태를 더 좋은 상태로 본다.

즉, 모델은 학습을 통해 데이터를 잘 설명할 수 있는 낮은 에너지 상태를 찾으려고 한다.

------

## 3. Energy-Based Model, EBM

Boltzmann Machine을 이해하려면 먼저 **Energy-Based Model**을 알아야 한다.

Energy-Based Model은 말 그대로 데이터나 상태에 **에너지 값을 부여하는 모델**이다.

- 좋은 데이터 상태 → 낮은 에너지
- 이상하거나 부자연스러운 상태 → 높은 에너지

라고 생각하면 된다.

즉, 모델은 학습을 통해 좋은 데이터는 낮은 에너지로 만들고, 좋지 않은 데이터는 높은 에너지로 만들도록 훈련된다.

------

## 4. RBM이 등장한 이유

원래 Boltzmann Machine은 매우 강력하지만 계산량이 너무 크다는 문제가 있다.

모든 뉴런이 서로 복잡하게 연결되어 있기 때문에 학습이 어렵고 시간이 오래 걸린다.

그래서 이를 단순화한 구조가 등장했는데, 그것이 바로 **Restricted Boltzmann Machine, RBM**이다.

Restricted는 “제한된”이라는 뜻이다.

즉, RBM은 기존 Boltzmann Machine에서 일부 연결을 제한해서 계산을 더 쉽게 만든 모델이다.

------

## 5. RBM 구조

RBM은 크게 두 층으로 구성된다.

- Visible Layer
- Hidden Layer

Visible Layer는 입력 데이터가 들어오는 층이다.

Hidden Layer는 입력 데이터의 숨겨진 특징을 학습하는 층이다.

중요한 점은 같은 층 안의 뉴런끼리는 연결되지 않는다는 것이다.

즉,

- Visible Layer 내부 연결 없음
- Hidden Layer 내부 연결 없음
- Visible Layer와 Hidden Layer 사이만 연결됨

이 구조 덕분에 기존 Boltzmann Machine보다 계산이 훨씬 쉬워진다.

------

## 6. Contrastive Divergence

RBM을 학습할 때 사용하는 대표적인 알고리즘이 **Contrastive Divergence**이다.

Contrastive Divergence는 RBM의 가중치를 학습하기 위한 방법이다.

간단히 말하면,

1. 실제 데이터를 넣는다
2. Hidden Layer가 특징을 학습한다
3. 다시 데이터를 재구성한다
4. 원래 데이터와 재구성된 데이터를 비교한다
5. 차이를 줄이는 방향으로 가중치를 수정한다

이 과정을 통해 RBM은 입력 데이터의 패턴을 점점 더 잘 학습하게 된다.

------

## 7. DBN과 DBM

마지막으로 간단히 다룰 개념은

- DBN: Deep Belief Network
- DBM: Deep Boltzmann Machine

이다.

이 둘은 Boltzmann Machine과 RBM을 확장한 더 깊고 복잡한 딥러닝 모델이다.

다만 이번 강의에서는 DBN과 DBM을 깊게 다루지는 않고, 간단한 개념과 참고 자료 정도만 소개한다.

------

## 8. 핵심 정리

- Boltzmann Machine은 확률 기반 신경망 모델이다.
- 핵심 개념은 **Energy**이다.
- 좋은 상태는 낮은 에너지를 가진다.
- Energy-Based Model은 데이터 상태에 에너지 값을 부여하는 모델이다.
- 원래 Boltzmann Machine은 계산량이 너무 크다.
- 이를 해결하기 위해 RBM이 등장했다.
- RBM은 Visible Layer와 Hidden Layer로 구성된다.
- RBM 학습에는 Contrastive Divergence가 사용된다.
- DBN과 DBM은 Boltzmann Machine 계열의 더 발전된 모델이다.

------

## 9. 추가

Boltzmann Machine을 처음 들으면 에너지라는 개념 때문에 어렵게 느껴질 수 있다.

하지만 핵심은 단순하다. 모델이 보기에 자연스럽고 맞는 데이터는 낮은 에너지를 가지게 만들고,

부자연스럽거나 틀린 데이터는 높은 에너지를 가지게 만드는 것이다.

즉, Boltzmann Machine은 데이터의 패턴을 외우는 것이 아니라 어떤 상태가 더 그럴듯한지를 에너지 관점에서 학습하는 모델이다.
