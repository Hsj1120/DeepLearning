# 📌 RNN과 LSTM 기반 주가 예측 - 개념 및 실습 정리

## 1. RNN(Recurrent Neural Network)이란?

RNN은 순서(sequence) 데이터를 처리하기 위한 딥러닝 모델이다.

기존 신경망(ANN)은 입력을 한 번 처리하면 끝나지만,
RNN은 이전 정보를 기억하면서 다음 계산에 활용한다.

즉,

- 이전 상태를 기억하고
- 시간 흐름(Time Sequence)을 따라 데이터를 이해하는 모델이다.

------

## 2. 왜 RNN이 필요한가?

일반 신경망은 데이터 간의 “순서 관계”를 이해하지 못한다.

하지만 실제 데이터는 대부분 순서가 중요하다.

예시:

- 문장
- 음성
- 주가
- 센서 데이터
- 자율주행 로그

이런 데이터는 이전 정보가 현재 판단에 영향을 준다.

예를 들어:

- “오늘”의 주가는 어제와 연관이 있다.
- 문장의 다음 단어는 이전 단어의 영향을 받는다.

따라서 시간 흐름을 이해하는 모델이 필요하며,
그 역할을 하는 것이 RNN이다.

------

# 3. 인간의 뇌와 비교한 RNN 직관

딥러닝은 인간의 뇌 구조에서 아이디어를 가져왔다.

## ✔ ANN vs CNN vs RNN

| 모델 | 역할                  | 뇌와 비교 |
| ---- | --------------------- | --------- |
| ANN  | 일반 패턴 학습        | 장기 기억 |
| CNN  | 이미지 처리           | 시각 처리 |
| RNN  | 순서/시간 데이터 처리 | 단기 기억 |

------

## ✔ 핵심 차이

### ANN

- 학습된 가중치를 저장
- 항상 동일한 방식으로 동작
- “장기 기억” 역할

### RNN

- 직전 상태를 기억
- 흐름을 따라 계산
- “단기 기억” 역할

------

# 4. RNN의 핵심 구조

기존 신경망:

입력 → 출력

RNN:

입력 → 출력 → 다시 자기 자신에게 전달

즉, 출력 일부가 다시 입력으로 들어간다.

이를:

> Temporal Loop (시간 루프)

라고 한다.

------

## ✔ 의미

RNN은 이전 상태(hidden state)를 유지하며 계산한다.

즉:

현재 결과 = 현재 입력 + 과거 기억

------

# 5. Unrolling (시간축 펼치기)

RNN은 실제로 시간축으로 펼쳐서 이해해야 한다.

## ✔ 구조

t=1 → t=2 → t=3 → ...

각 시점마다 동일한 네트워크가 반복된다.

하지만 중요한 점은:

이전 시점의 결과가 다음 시점에 전달된다는 것이다.

------

## ✔ 핵심 정리

RNN은:

- 시간을 따라 연결되고
- 이전 정보를 계속 전달하며
- 문맥(Context)을 이해한다.

------

# 6. RNN의 대표 구조 유형

## ① One-to-Many

하나 입력 → 여러 출력

예시:

- 이미지 캡션 생성
- 음악 생성

------

## ② Many-to-One

여러 입력 → 하나 출력

예시:

- 감정 분석
- 스팸 분류

문장 전체를 보고 최종 결과를 판단한다.

------

## ③ Many-to-Many

여러 입력 → 여러 출력

예시:

- 번역
- 자막 생성
- 음성 인식

------

# 7. RNN의 한계

기본 RNN은 긴 문맥(Long-Term Dependency)을 잘 기억하지 못한다.

문제가 되는 이유:

- 앞쪽 정보가 점점 사라짐
- 긴 문장이나 긴 시계열에서 성능 저하 발생

이를:

> Vanishing Gradient Problem

이라고 한다.

------

# 8. 그래서 등장한 것이 LSTM

LSTM(Long Short-Term Memory)은
RNN의 기억 문제를 해결하기 위해 등장한 모델이다.

------

## ✔ 특징

LSTM은:

- 중요한 정보는 유지
- 불필요한 정보는 삭제
- 장기 의존성을 더 잘 학습

할 수 있다.

즉,

기존 RNN보다 훨씬 긴 흐름을 기억할 수 있다.

------

> 미래 주가 변화는 과거와 독립적이다

라는 Brownian Motion 개념이 존재한다.

즉:

정확한 미래 가격을 맞추는 것은 사실상 불가능하다.

하지만:

- 전체적인 흐름
- 상승/하락 패턴

은 일부 학습 가능하다.

RNN의 핵심은 단순히 “반복 구조”가 아니다.

- 과거 정보를 기억하고
- 현재 판단에 활용하며
- 시간 흐름을 이해한다는 점이다.

------

# 🎯 최종 핵심 정리

- RNN은 순서 데이터를 처리하는 딥러닝 모델이다.
- 이전 상태를 기억하며 계산한다.
- 시간 흐름과 문맥(Context)을 이해할 수 있다.
- 긴 기억 문제를 해결하기 위해 LSTM이 등장했다.
- LSTM은 주가 예측, 번역, 음성 인식 등 다양한 분야에서 활용된다.
