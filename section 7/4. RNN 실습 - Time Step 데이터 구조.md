# 📌 RNN 데이터 전처리 - Time Step 데이터 구조 생성

## 1. 이번 단계의 핵심 목표

이번 단계는 RNN에서 가장 중요한 부분 중 하나다.

이전까지는:

- CSV 불러오기
- 필요한 컬럼 선택
- 정규화(Normalization)

까지만 수행했다.

이번에는:

> RNN이 “과거 데이터를 기억하면서” 학습할 수 있도록
> 특별한 입력 구조(Time Step 구조)를 만든다.



------

# 2. Time Step이란?

RNN은 일반 딥러닝과 다르게:

> “과거 정보”를 기억하면서 학습

한다.

예를 들어:

```python
어제 주가만 보고 내일 주가 예측
```

하는 게 아니라,

```python
최근 60일 주가 흐름을 보고
내일 주가 예측
```

하는 방식이다.

------

# 3. 왜 과거 데이터를 여러 개 보는가?

주가는 단순히 하루 데이터만으로 움직이지 않는다.

예를 들어:

| 날짜 | 주가 |
| ---- | ---- |
| 1일  | 100  |
| 2일  | 105  |
| 3일  | 110  |
| 4일  | 115  |

이렇게 계속 상승 중이라면:

> “상승 추세(trend)”

를 학습할 수 있다.

즉 RNN은:

```python
숫자 하나
```

가 아니라

```python
시간 흐름 패턴
```

을 학습하는 모델이다.

------

# 4. 왜 60 Time Step을 사용하는가?

이번 실습에서는:

```python
60 timesteps
```

를 사용한다.

즉:

> 최근 60일 데이터를 보고 다음 하루를 예측

한다.

------

## ✔ 실제 의미

금융 시장 기준:

```python
한 달 ≈ 20 거래일
```

이므로:

```python
60일 ≈ 3개월
```

이다.

즉:

> 최근 3개월 흐름을 보고 내일 주가를 예측

하는 것이다.

------

# 5. 입력(X_train)과 출력(y_train)

## ✔ X_train

입력 데이터

```python
과거 60일 주가
```

저장

------

## ✔ y_train

정답 데이터(Label)

```python
다음 날 주가
```

저장

------

# 6. 실제 구조 예시

예를 들어 주가가:

```python
[1,2,3,4,5,6]
```

라고 해보자.

------

## timestep = 3 이라면

### 첫 번째 입력

```python
X = [1,2,3]
```

### 첫 번째 정답

```python
y = 4
```

즉:

```python
1,2,3 흐름을 보고
4를 예측
```

학습하는 것이다.

------

## 다음 데이터

### 입력

```python
X = [2,3,4]
```

### 정답

```python
y = 5
```

즉:

```python
2,3,4 흐름 → 5 예측
```

------

# 7. Sliding Window 개념

이 구조를:

# ✔ Sliding Window

라고 부른다.

------

## 그림 느낌

```python
[1 2 3] -> 4
  [2 3 4] -> 5
    [3 4 5] -> 6
```

창(window)이 한 칸씩 이동한다.

------

## 빈 리스트 생성

```python
X_train = []
y_train = []
```

------

## 왜 리스트로 시작?

데이터를 반복문으로 계속 추가해야 하기 때문이다.

# 10. 왜 60부터 시작하는가?

처음 60개 데이터가 있어야:

```python
과거 60일
```

을 만들 수 있기 때문이다.

------

## 예시

```python
i = 60
```

이면:

과거 데이터:

```python
0 ~ 59
```

사용 가능

------

## 그런데

```python
i = 10
```

이면:

과거 60개 데이터가 존재하지 않는다.

------

# 11. X_train 생성

## 코드

```python
X_train.append(training_set_scaled[i-60:i, 0])
```

------

# 12. 이 코드 해석

## 핵심

```python
i-60 : i
```

이다.

------

## 예시

```python
i = 60
```

이면:

```python
0:60
```

이 된다.

------

## 의미

```python
0~59번째 데이터
```

가져오기

------

# 13. 왜 60개인가?

파이썬 슬라이싱은:

```python
끝 번호 제외
```

하기 때문이다.

즉:

```python
0:60
```

은:

```python
0~59
```

총 60개다.

------

# 14. 마지막의 0 의미

```python
training_set_scaled[i-60:i, 0]
```

에서:

```python
0
```

은:

```python
0번 컬럼(Open Price)
```

의 의미다.

------

# 15. y_train 생성

## 코드

```python
y_train.append(training_set_scaled[i, 0])
```

------

# 20. 최종 데이터 구조

## 의미

| 값   | 의미             |
| ---- | ---------------- |

------

# 21. 왜 1198개인가?

원본 데이터:

```python
1258개
```

있다.

------

하지만:

```python
처음 60개는
과거 데이터 부족
```

때문에 사용 못 한다.

즉:

```python
1258 - 60 = 1198
```

이다.

------

# 22. 이번 단계의 핵심 목적

이번 단계의 진짜 목적은:

> “단순 숫자”를
> “시간 흐름 데이터”로 바꾸는 것

이다.

즉:

```python
[과거 60일] -> 다음 하루
```

형태로 변환한 것이다.

------

# 🎯 최종 핵심 정리

- RNN은 시간 흐름 데이터를 학습한다.
- 따라서 과거 데이터를 묶어서 입력해야 한다.
- 이번 실습에서는 최근 60일 데이터를 사용한다.
- `X_train` 은 과거 60일 데이터
- `y_train` 은 다음 날 주가
- Sliding Window 방식으로 데이터 생성
- 마지막에 NumPy 배열로 변환

------
