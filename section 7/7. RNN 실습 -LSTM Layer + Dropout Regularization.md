# 📌 Part 2-2: 첫 번째 LSTM Layer 추가 + Dropout Regularization

이번 단계에서는 드디어:

> 실제 첫 번째 LSTM Layer를 RNN에 추가

하게 된다.

그리고 함께:

> Dropout Regularization

도 추가해서 과적합(Overfitting)을 방지한다.

------

# 1. 이번 단계에서 하는 일

이번 단계 핵심은 딱 2가지이다.

## ✔ 첫 번째 LSTM Layer 추가

```python
regressor.add(LSTM(...))
```

------

## ✔ Dropout 추가

```python
regressor.add(Dropout(...))
```

------

# 2. 전체 코드

## 코드

```python
regressor.add(
    LSTM(
        units = 50,
        return_sequences = True,
        input_shape = (X_train.shape[1], 1)
    )
)

regressor.add(Dropout(0.2))
```

------

# 3. add() 메서드란?

우리가 만든:

```python
regressor = Sequential()
```

는:

> layer들을 순서대로 저장하는 객체

였다.

------

## 따라서

```python
regressor.add(...)
```

는:

> 신경망에 새로운 층(layer)을 추가

하는 함수이다.

------

# 4. 첫 번째 add()

## 코드

```python
regressor.add(LSTM(...))
```

의 의미는:

```python
RNN에 LSTM layer 추가
```

이다.

------

# 5. LSTM() 안의 핵심 인자 3개

강의에서 아주 중요하다고 강조한 부분이다.

## 핵심 인자

| 인자               | 의미                           |
| ------------------ | ------------------------------ |
| `units`            | LSTM 뉴런 개수                 |
| `return_sequences` | 다음 LSTM에 sequence 전달 여부 |
| `input_shape`      | 입력 데이터 구조               |

------

# 6. units = 50

## 코드

```python
units = 50
```

------

# 7. units란?

LSTM 안의:

```python
memory cell (기억 셀)
```

개수이다.

강의에서는 이해 쉽게 그냥:

```python
뉴런(neuron)
```

이라고 설명한다.

------

# 8. 왜 50개를 사용하는가?

주가 예측은 매우 복잡하다.

왜냐하면:

- 상승
- 하락
- 장기 패턴
- 단기 변동

등을 모두 학습해야 하기 때문이다.

------

## 뉴런 수가 너무 적으면?

예:

```python
units = 3
```

이면:

```python
복잡한 패턴 학습 불가능
```

------

## 따라서

```python
50개 정도 사용
```

하여:

> 높은 차원(high dimensionality)의 표현

을 만들게 된다.

------

# 9. high dimensionality란?

쉽게 말하면:

> 더 복잡한 특징을 표현할 수 있는 공간

이다.

------

## 느낌

뉴런이 많을수록:

```python
더 다양한 패턴 기억 가능
```

------

# 10. return_sequences=True

## 코드

```python
return_sequences = True
```

------

# 11. 이게 가장 중요한 부분 중 하나

LSTM은 출력할 때 2가지 방식이 있다.

------

# 12. return_sequences=False

(default)

## 출력

```python
마지막 timestep 결과만 출력
```

------

## 예시

입력:

```python
Day1 Day2 Day3 Day4
```

출력:

```python
Day4 결과만 출력
```

------

# 13. return_sequences=True

## 출력

```python
모든 timestep 결과 반환
```

------

## 예시

입력:

```python
Day1 Day2 Day3 Day4
```

출력:

```python
h1 h2 h3 h4
```

------

# 14. 왜 True로 해야 하나?

우리는:

```python
Stacked LSTM
```

을 만들고 있다.

즉:

```python
LSTM
 ↓
LSTM
 ↓
LSTM
```

처럼 다음 LSTM layer가 또 존재한다.

------

# 15. 다음 LSTM은 sequence 전체가 필요하다

다음 층도:

```python
시간 흐름(sequence)
```

을 받아야 한다.

따라서:

```python
return_sequences=True
```

로:

```python
모든 timestep 출력
```

을 넘겨줘야 한다.

------

# 16. 만약 False면?

```python
마지막 값 하나만 전달
```

되므로:

```python
다음 LSTM layer 사용 불가능
```

------

# 17. 마지막 LSTM만 False

보통:

```python
중간 LSTM들:
return_sequences=True
```

------

마지막 LSTM:

```python
return_sequences=False
```

를 사용한다.

왜냐하면 마지막에는:

```python
출력층(Dense)
```

으로 가기 때문이다.

------

# 18. input_shape

## 코드

```python
input_shape = (X_train.shape[1], 1)
```

------

# 19. 원래 X_train shape 복습

우리가 만든 입력 구조:

```python
(1198, 60, 1)
```

------

## 의미

| 값   | 의미        |
| ---- | ----------- |
| 1198 | sample 개수 |
| 60   | timestep    |
| 1    | feature     |

------

# 20. 그런데 input_shape는 왜 2개만 쓰는가?

Keras는:

```python
sample 개수는 자동 처리
```

한다.

따라서:

```python
(timesteps, features)
```

만 넣는다.

------

# 21. 실제 입력 의미

## 코드

```python
input_shape = (60, 1)
```

의 의미:

```python
60일 시계열
feature는 1개(Open Price)
```

------

# 22. 이제 Dropout 추가

## 코드

```python
regressor.add(Dropout(0.2))
```

------

# 23. Dropout이란?

Dropout은:

> 일부 뉴런을 랜덤하게 끄는 기법

이다.

------

# 24. 왜 사용하는가?

딥러닝은 너무 학습을 잘하면:

```python
훈련 데이터 암기
```

를 해버린다.

이를:

```python
Overfitting
```

이라고 한다.

------

# 25. Dropout의 역할

훈련 중:

```python
일부 뉴런 무시
```

하게 만들어:

```python
특정 뉴런 의존 감소
```

효과를 준다.

------

# 26. Dropout(0.2)

## 의미

```python
20% 뉴런 제거
```

------

현재:

```python
units = 50
```

이므로:

```python
50 × 0.2 = 10
```

------

즉:

```python
10개 뉴런 랜덤 비활성화
```

------

# 27. 언제 제거되는가?

훈련 중:

- Forward Propagation
- Backpropagation

과정에서 랜덤하게 제거된다.

------

# 28. 실제 구조 느낌

현재까지 구조:

```python
Input
 ↓
LSTM(50)
 ↓
Dropout(20%)
```

------

# 29. 앞으로 어떻게 되는가?

다음 단계에서:

```python
LSTM + Dropout
```

를 여러 번 더 추가한다.

최종적으로:

```python
4개의 LSTM layer
```

를 쌓게 된다.

------

# 30. 현재까지 핵심 흐름

## 지금까지 만든 구조

```python
Sequential()
 ↓
첫 번째 LSTM
 ↓
Dropout
```

------

# 🎯 최종 핵심 정리

- `units=50`
  → LSTM 뉴런 50개 사용
- `return_sequences=True`
  → 다음 LSTM layer에 sequence 전체 전달
- `input_shape=(60,1)`
  → 60 timestep + feature 1개
- `Dropout(0.2)`
  → 뉴런 20% 랜덤 제거
- Dropout은 과적합 방지 목적

------

# 🔥 한 줄 최종 정리

> 이번 단계는 첫 번째 LSTM layer(50개 뉴런)를 추가하고, 다음 LSTM layer로 sequence를 전달하기 위해 `return_sequences=True`를 설정하며, Dropout(20%)으로 과적합을 방지하는 과정이다.
