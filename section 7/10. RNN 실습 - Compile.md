# 📌 Part 2-5: RNN Compile 하기 (Optimizer + Loss 설정)

이번 단계에서는:

> 우리가 만든 RNN이 “어떻게 학습할지”

를 설정한다.

즉:

```python
regressor.compile(...)
```

을 통해:

- 가중치를 어떻게 업데이트할지
- 오차를 어떻게 계산할지

정의하는 단계이다.

------

# 1. 이번 단계 코드

## 코드

```python
regressor.compile(
    optimizer = 'adam',
    loss = 'mean_squared_error'
)
```

------

# 2. compile이란?

compile은:

> 모델의 학습 방법을 설정하는 과정

이다.

------

## 쉽게 말하면

```python
"이 신경망을 어떤 방식으로 학습시킬까?"
```

를 정하는 단계이다.

------

# 3. compile에서 가장 중요한 2개

| 요소      | 역할                 |
| --------- | -------------------- |
| optimizer | 가중치 업데이트 방법 |
| loss      | 오차 계산 방법       |

------

# 4. optimizer란?

optimizer는:

> 모델이 어떻게 정답 방향으로 학습할지 결정

하는 알고리즘이다.

------

# 5. 딥러닝 학습의 핵심

딥러닝은 결국:

```python
예측값
```

과

```python
실제값
```

차이를 줄이는 과정이다.

------

# 6. 그런데 어떻게 줄일까?

예를 들어:

```python
현재 예측: 100
실제값: 120
```

이면:

```python
오차 = 20
```

이 발생한다.

------

그러면 optimizer가:

```python
"weight를 어떻게 수정해야 오차가 줄어들까?"
```

를 계산한다.

------

# 7. 즉 optimizer 역할

```python
weight 업데이트 전략
```

이다.

------

# 8. 현재 사용한 optimizer

## 코드

```python
optimizer = 'adam'
```

------

# 9. Adam Optimizer란?

Adam은:

> 가장 많이 사용하는 딥러닝 optimizer 중 하나

이다.

------

# 10. 왜 Adam을 많이 쓰는가?

Adam은:

- 학습 속도 빠름
- 안정적
- 자동으로 learning rate 조절
- 대부분 문제에서 성능 좋음

특징이 있다.

------

# 11. 강의에서 언급한 RMSprop

강의에서는:

```python
RMSprop
```

도 소개한다.

------

# 12. RMSprop이란?

RMSprop은:

> RNN 계열에서 자주 사용하는 optimizer

이다.

특히:

```python
시계열(sequence)
```

데이터에서 성능이 좋은 편이다.

------

# 13. 그런데 왜 Adam 사용?

강의에서는 실험 결과:

```python
Adam이 더 좋은 결과
```

를 보여서 Adam을 사용한다고 설명한다.

------

# 14. 실제로 Adam은 거의 기본 선택

현업이나 연구에서도:

```python
Adam = 안전한 기본 선택
```

느낌이다.

------

# 15. 이제 loss 함수

## 코드

```python
loss = 'mean_squared_error'
```

------

# 16. loss란?

loss는:

> 모델이 얼마나 틀렸는지 계산하는 함수

이다.

------

# 17. 왜 필요한가?

딥러닝은:

```python
얼마나 틀렸는지
```

를 알아야 학습할 수 있다.

------

즉:

```python
오차 계산
```

이 반드시 필요하다.

------

# 18. 현재 문제는 Regression

우리는 지금:

```python
주가 예측
```

을 하고 있다.

즉:

```python
연속적인 숫자 값 예측
```

이다.

------

# 19. Regression에서 대표 Loss

회귀 문제에서는 보통:

```python
MSE(Mean Squared Error)
```

를 사용한다.

------

# 20. Mean Squared Error란?

이름 그대로:

```python
오차 제곱 평균
```

이다.

------

# 21. 공식

![7-3](..\img\7-3.png)

------

# 22. 의미 해석

| 기호        | 의미   |
| ----------- | ------ |
| (y_i)       | 실제값 |
| (\hat{y}_i) | 예측값 |

------

즉:

```python
(실제값 - 예측값)^2
```

를 평균낸 것이다.

------

# 23. 왜 제곱을 할까?

제곱하는 이유:

- 음수 제거
- 큰 오차를 더 크게 벌점

주기 위해서이다.

------

# 24. 예시

## 예측

```python
예측값 = 100
실제값 = 120
```

------

## 오차

```python
120 - 100 = 20
```

------

## 제곱

```python
20² = 400
```

------

# 25. 큰 오차를 강하게 벌점

예를 들어:

| 오차 | 제곱 |
| ---- | ---- |
| 2    | 4    |
| 10   | 100  |

------

즉:

```python
큰 실수는 훨씬 크게 처벌
```

한다.

------

# 26. 왜 classification과 다를까?

이전 ANN/CNN에서는:

```python
binary_crossentropy
```

를 사용했다.

------

왜냐하면 그때는:

```python
분류(classification)
```

문제였기 때문이다.

------

하지만 지금은:

```python
회귀(regression)
```

이므로:

```python
MSE 사용
```

한다.

------

즉:

# 🎯 최종 핵심 정리

- `compile()`은 학습 방식 설정 단계

- `optimizer='adam'`
  → weight 업데이트 방법

- Adam은 가장 많이 사용하는 optimizer 중 하나

- `loss='mean_squared_error'`
  → 회귀 문제용 loss 함수

- MSE는:

  ```python
  (실제값 - 예측값)^2 평균
  ```

- 다음 단계는:

  ```python
  fit()
  ```

  으로 실제 학습 시작

------

# 🔥 한 줄 최종 정리

> 이번 단계는 Adam optimizer를 사용해 가중치 업데이트 방식을 설정하고, Mean Squared Error(MSE)를 loss 함수로 사용하여 “예측 주가와 실제 주가의 차이”를 최소화하도록 RNN 학습 방식을 정의하는 과정이다.
