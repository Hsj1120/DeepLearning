# 📌 Part 2: Building the RNN — RNN 구조 만들기

이번 파트부터는 본격적으로:

> 실제 RNN(Recurrent Neural Network) 모델 구조를 만드는 단계

에 들어간다.

여기서는 단순한 LSTM 하나만 만드는 것이 아니라:

- 여러 층의 LSTM을 쌓은(Stacked LSTM)
- 과적합 방지를 위한 Dropout까지 포함한

좀 더 강력한 RNN 구조를 만들게 된다.

------

# 1. 이번 단계에서 사용할 Keras 클래스들

강의에서는 먼저 필요한 라이브러리들을 import 한다.

## 코드

```python
from keras.models import Sequential
from keras.layers import Dense
from keras.layers import LSTM
from keras.layers import Dropout
```

------

# 2. 각각 무슨 역할인가?

| 클래스       | 역할                             |
| ------------ | -------------------------------- |
| `Sequential` | 층(layer)을 순서대로 쌓는 신경망 |
| `Dense`      | 출력층(Output Layer)             |
| `LSTM`       | LSTM 레이어                      |
| `Dropout`    | 과적합 방지                      |

------

# 3. Sequential이란?

## 코드

```python
regressor = Sequential()
```

------

## 의미

`Sequential`은:

> 층(layer)을 순서대로 쌓는 신경망 구조

를 의미한다.

즉:

```python
입력층
 ↓
LSTM
 ↓
LSTM
 ↓
Dense
```

처럼 층을 차례대로 연결하는 방식이다.

------

# 4. 왜 이름이 regressor일까?

이전 ANN/CNN에서는:

```python
classifier
```

라는 이름을 사용했다.

하지만 이번에는:

```python
regressor
```

라는 이름을 사용한다.

------

# 5. Classifier vs Regressor

## ✔ Classifier

분류(Classification)

예:

- 고양이 / 강아지
- 스팸 / 정상 메일

처럼:

> “클래스(category)”를 예측

하는 모델

------

## ✔ Regressor

회귀(Regression)

예:

- 주가
- 온도
- 판매량

처럼:

> 연속적인 숫자 값(continuous value)

을 예측하는 모델

------

# 6. 이번 문제는 회귀 문제

우리가 지금 예측하는 것은:

```python
Google 주가
```

이다.

주가는:

```python
523.21
841.52
```

처럼 연속적인 실수 값이다.

따라서:

```python
Regression 문제
```

가 된다.

그래서 이름도:

```python
regressor
```

라고 지은 것이다.

------

# 7. Sequential 내부 느낌

## 코드

```python
regressor = Sequential()
```

이 실행되면 내부적으로는:

```python
빈 신경망 객체 생성
```

상태가 된다.

아직 아무 층도 없다.

즉:

```python
[]
```

같은 느낌이다.

------

# 8. 이후 어떻게 되는가?

이제 앞으로:

```python
regressor.add(...)
```

를 통해 층을 하나씩 추가하게 된다.

예:

```python
regressor.add(LSTM(...))
regressor.add(Dropout(...))
```

------

# 9. Stacked LSTM이란?

이번 강의에서 만들 구조는:

> LSTM을 여러 층 쌓은 구조

이다.

------

## 일반 LSTM

```python
Input
 ↓
LSTM
 ↓
Output
```

------

## Stacked LSTM

```python
Input
 ↓
LSTM
 ↓
LSTM
 ↓
LSTM
 ↓
Output
```

------

# 10. 왜 여러 층을 쌓을까?

층을 여러 개 쌓으면:

- 더 복잡한 패턴 학습 가능
- 장기 시계열 패턴 학습 강화
- 고차원 특징 추출 가능

해진다.

즉:

> 더 깊은 시계열 특징을 학습

할 수 있게 된다.

------

# 11. Dropout은 왜 사용하는가?

딥러닝 모델은 너무 학습을 잘하면:

```python
훈련 데이터만 외워버리는 현상
```

이 발생할 수 있다.

이를:

```python
Overfitting (과적합)
```

이라고 한다.

------

# 12. Dropout의 역할

Dropout은 학습 중 일부 뉴런을 랜덤하게 끈다.

예:

```python
뉴런 20% 비활성화
```

------

## 효과

모델이:

```python
특정 뉴런 하나에만 의존하지 못하게 함
```

→ 일반화 성능 향상

------

# 13. LSTM + Dropout 느낌

```python
Input
 ↓
LSTM
 ↓
Dropout
 ↓
LSTM
 ↓
Dropout
 ↓
Dense
```

------

# 14. 강의에서 말하는 중요한 포인트

강의에서:

> RNN을 “sequence of layers”로 만든다

라고 설명한다.

------

## sequence of layers 의미

즉:

```python
층을 순서대로 연결한 구조
```

라는 뜻이다.

------

# 15. Sequential vs Computational Graph

강의에서는 잠깐:

```python
Computational Graph
```

이야기도 나온다.

------

## Sequential

단순 직선 구조

```python
A → B → C
```

------

## Computational Graph

복잡한 연결 가능

```python
      → B →
A →         → D
      → C →
```

------

# 16. 왜 지금은 Sequential을 쓰는가?

현재 만드는 모델은:

```python
순차적 구조
```

이므로 Sequential만으로 충분하다.

즉:

```python
입력 → LSTM → 출력
```

처럼 단순하게 이어진다.

------

# 17. Keras가 RNN 구현에 좋은 이유

Keras는:

- 코드가 짧고
- 직관적이며
- 빠르게 실험 가능

해서 RNN 입문에 매우 좋다.

------

# 18. 현재까지 완성된 상태

현재 코드:

```python
regressor = Sequential()
```

만 실행된 상태는:

```python
빈 RNN 모델 생성 완료
```

상태이다.

아직:

- LSTM 없음
- 출력층 없음
- Dropout 없음

------

# 19. 앞으로 추가될 것들

다음 단계에서:

- 첫 번째 LSTM
- Dropout
- 추가 LSTM
- Dense 출력층

을 하나씩 추가하게 된다.

------

# 🎯 최종 핵심 정리

- `Sequential()`은 층을 순서대로 쌓는 신경망 객체
- 이번 문제는 주가 예측이므로 Regression 문제
- 따라서 모델 이름을 `regressor`로 사용
- 이번에는 단순 LSTM이 아니라:
  - Stacked LSTM
  - Dropout Regularization
    을 함께 사용
- Dropout은 과적합 방지 역할

------

# 🔥 한 줄 최종 정리

> 이번 단계는 `Sequential()`을 이용해 “층을 순서대로 쌓는 RNN 틀(regressor)”을 생성하고, 이후 Stacked LSTM과 Dropout을 추가할 준비를 하는 과정이다.

업로드한 기존 정리와 흐름 맞춰서 설명했어.
