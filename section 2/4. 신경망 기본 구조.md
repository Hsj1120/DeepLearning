------

## 1. 신경망 기본 구조

- 신경망은
  **입력값 → 가중합 → 활성화 함수 → 은닉층 → 출력층** 구조로 동작함
- 여기서 중요한 것은
  -> **은닉층이 새로운 의미 있는 특징을 만들어낸다는 점**

------

## 2. 입력층 (Input Layer)

- 예시 입력값:
  - 면적
  - 방 개수
  - 도시와의 거리
  - 건물 나이
- 특징:
  - 데이터 그대로 들어옴
  - 아직 가공되지 않은 상태

------

## 3. 은닉층 (Hidden Layer)

### ① 기본 개념

- 각 뉴런은

  -> **특정 패턴을 감지하는 역할**

- 입력값 전체를 다 쓰는 게 아니라

  ->**일부 조합만 선택해서 사용**

------

### ② 동작 방식

각 뉴런은:

- 입력값 일부 선택
- 가중합 계산
- 특정 조건 만족 시 활성화

👉 즉, **조건 기반 필터처럼 동작**

------

### ③ 예시

- (면적 + 거리)
  → 가까운데 큰 집 = 가치 높음
- (면적 + 방 개수 + 나이)
  → 넓고 방 많고 신축 = 가족 선호
- (나이만 사용)
  → 100년 이상 = 역사적 가치

------

### ④ 핵심 특징

- 여러 입력을 조합해서
  -> **새로운 특징(feature)** 생성
- 사람이 직접 정의하지 않아도
  -> **자동으로 패턴 학습**

------

## 4. 활성화(Activation)의 의미

- 뉴런은 조건을 만족하면 “활성화됨”
- 예:
  - 조건 만족 → 값 전달
  - 조건 불만족 → 0 또는 매우 작은 값

-> 그래서 “activation function”이라고 부름

------

## 5. 출력층 (Output Layer)

- 은닉층에서 만든 특징들을 종합해서
  -> 최종 결과 생성
- 예:
  - 집 가격 예측
  - 분류 결과 (0 또는 1)

------

## 6. 왜 은닉층이 중요한가?

- 단순 모델:
  - 입력 → 출력
    → 복잡한 관계 표현 어려움
- 신경망:
  - 입력 → 은닉층 → 출력
    → 다양한 패턴 조합 가능

-> 복잡한 문제 해결 가능

------

## 7. 핵심 정리

- 뉴런 = **패턴 감지기**

- 은닉층 = **특징 생성기**

- 출력층 = **최종 판단**

- 신경망은:

  -> **여러 뉴런이 협력해서 복잡한 문제를 해결하는 구조**

------

## 8. 한 줄 핵심

-> **신경망은 입력 데이터를 조합해서 새로운 특징을 만들고, 그걸 기반으로 최종 결과를 예측하는 구조다**

------

## 9. 추가 (직관 포인트)

- 뉴런 하나 → 단순한 판단
- 여러 뉴런 → 복잡한 패턴 이해
