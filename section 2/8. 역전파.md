# 1. 전체 흐름 (핵심 구조)

👉 신경망 학습 과정:

```
입력 → 예측(ŷ) → 오차 계산 → 역전파 → 가중치 수정 → 반복
```

------

# 2. 핵심 개념 3가지

## ① 순전파

- 입력 데이터 → 출력(ŷ) 생성

------

## ② 비용 함수

- 실제값(y) vs 예측값(ŷ)
- 오차 계산

👉 목표:
**오차 최소화**

------

## ③ 역전파

- 오차를 뒤로 전달
- 각 가중치의 영향 계산
- **모든 가중치를 동시에 업데이트**

👉 핵심 특징

------

# 3. 학습 과정 

## Step 1. 가중치 초기화

- 랜덤 값으로 시작

------

## Step 2. 입력 데이터 넣기

- 한 row 입력

------

## Step 3. Forward Propagation

```
입력 → 계산 → ŷ
```

------

## Step 4. 오차 계산

```
오차 = (y - ŷ)
```

------

## Step 5. Backpropagation

- 오차를 뒤로 전달
- 각 가중치의 기여도 계산

------

## Step 6. 가중치 업데이트

```
W = W - (learning rate × gradient)
```

------

## Step 7. 반복

- 계속 반복 → 점점 정확해짐

------

# 4. SGD vs Batch 연결

## SGD

```
데이터 1개 → 바로 업데이트
```

## Batch

```
전체 데이터 → 한 번 업데이트
```

👉 차이:
**업데이트 타이밍**

------

# 5. Epoch 개념

👉 전체 데이터 1번 사용 = 1 epoch

- 여러 epoch 반복하면서 학습

------

# 6. 핵심 포인트 정리

- 가중치는 랜덤으로 시작
- 예측 → 오차 → 수정 반복
- Backpropagation으로 가중치 영향 계산
- 모든 가중치 동시에 업데이트
- 목표: Cost Function 최소화

------

# 7. 한 줄 핵심 정리

👉
**역전파는 오차를 기반으로 모든 가중치를 동시에 최적화하는 과정**
