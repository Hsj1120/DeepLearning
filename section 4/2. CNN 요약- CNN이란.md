# Convolutional Neural Network (CNN) 개념 이해 

CNN을 이해하기 위해서는 먼저
👉 **사람이 이미지를 어떻게 인식하는지**를 이해하는 것이 중요하다.

------

# 1. 사람은 이미지를 “특징”으로 인식한다

우리는 이미지를 볼 때 단순히 전체를 보는 것이 아니라,
👉 **특정한 특징(feature)**을 기준으로 판단한다.

예를 들어 아래 이미지에서

![4-1](..\img\section4\4-1.png)

- 어떤 사람은 “오른쪽을 보는 사람”을 보고
- 어떤 사람은 “정면을 보는 사람”을 본다

이 차이는 무엇일까?

👉 우리가 **어떤 특징을 먼저 보느냐**에 따라
이미지를 다르게 해석하기 때문이다

------

또 다른 예로

- 젊은 여자 vs 늙은 여자

![4-2](..\img\section4\4-2.png)

- 오리 vs 토끼

![4-3](..\img\section4\4-3.png)



같은 이미지에서도
👉 **어떤 특징을 선택하느냐에 따라 결과가 바뀐다**

------

### ✔ 핵심

- 인간의 뇌는 이미지를 볼 때
  → “전체”가 아니라
  → **특징을 기반으로 판단한다**

👉 한 줄 정리
→ “이미지는 특징(feature)의 조합으로 인식된다”

------

# 2. CNN도 사람과 비슷하게 동작한다

CNN이 하는 일도 결국 동일하다.

👉 **이미지에서 특징을 찾아서 분류하는 것**

즉,

- 눈, 코, 귀 같은 특징을 찾고
- 그것들을 조합해서
- “이건 고양이다”라고 판단한다

👉 사람과 거의 동일한 방식이다

------

# 3. 실제 CNN은 어떻게 판단할까?

CNN은 결과를 딱 하나로만 말하지 않는다.

대신 이런 식으로 말한다:

- 치타: 95%
- 표범: 3%
- 고양이: 2%

👉 즉, **확률 기반으로 판단**한다

------

### ✔ 중요한 포인트

- 확실한 경우 → 확률이 높음
- 애매한 경우 → 여러 후보가 비슷함

예를 들어 흐릿한 이미지에서는
👉 사람처럼 CNN도 헷갈린다

------

👉 한 줄 정리
→ “CNN도 사람처럼 확신이 없으면 여러 가능성을 둔다”

------

# 4. CNN이 중요한 이유

CNN은 현재 다양한 분야에서 사용된다.

예:

- 자율주행 → 사람, 신호 인식
- 페이스북 → 얼굴 자동 태깅
- 감정 인식 → 표정 분석

👉 즉, **이미지를 이해해야 하는 모든 분야에서 핵심 기술**

------

# 5. 이미지의 본질 (컴퓨터 관점)

우리가 보는 이미지는 사실
👉 컴퓨터 입장에서는 단순한 숫자이다

------

## ✔ 흑백 이미지

- 2차원 배열 (행렬)
- 각 픽셀 값: 0 ~ 255

![4-4](..\img\section4\4-4.png)

```id="m1b1dq"
0 → 검정
255 → 흰색
```

👉 중간 값 → 회색

------

## ✔ 컬러 이미지

- 3차원 배열

구성:

- Red (R)
- Green (G)
- Blue (B)

![4-5](..\img\section4\4-5.png)

👉 각 픽셀 = (R, G, B) 값

------

### ✔ 핵심

- 이미지는 결국
  → **숫자들의 집합**

![4-6](..\img\section4\4-6.png)

👉 CNN은 이 숫자를 가지고 학습한다

------

# 6. CNN 학습의 시작점

CNN은 처음부터 복잡한 이미지를 이해하지 않는다.

👉 가장 기본은 아주 단순한 이미지부터 시작한다

예:

- 2×2 이미지
- 0과 1로만 구성된 이미지

이렇게 단순화해서 원리를 이해하고
→ 점점 복잡한 이미지로 확장한다

------

# 7. CNN의 전체 처리 과정

CNN은 다음 단계로 이미지를 처리한다.

```
Convolution → MAX Pooling → Flattening → Fully Connected
```

👉 지금은 의미 몰라도 괜찮다

이 과정은 결국
👉 **특징을 찾고 → 줄이고 → 판단하는 흐름**이다

------

# 8. 핵심 개념 정리

### ✔ CNN의 본질

- 이미지를 숫자로 변환
- 특징을 추출
- 특징을 조합하여 분류

------

### ✔ 사람 vs CNN

| 사람             | CNN                 |
| ---------------- | ------------------- |
| 특징을 보고 판단 | 특징을 추출 후 판단 |
| 헷갈리면 고민    | 확률로 표현         |
| 경험으로 학습    | 데이터로 학습       |

------

# 한 줄 정리

👉 **“CNN은 이미지에서 특징을 찾아내고, 그 특징을 기반으로 확률적으로 판단하는 모델이다.”**
