# Convolutional Neural Network (CNN) 구조와 동작 원리

CNN(합성곱 신경망)은 이미지 데이터를 효과적으로 처리하기 위해 만들어진 신경망이다.
일반적인 신경망이 단순히 숫자를 처리하는 데 집중한다면, CNN은 이미지 속에 담긴 **공간적인 정보와 패턴**을 이해하는 데 특화되어 있다.

즉, CNN의 핵심 목적은
👉 **이미지에서 중요한 특징을 자동으로 추출하고, 이를 기반으로 무엇인지 판단하는 것**이다.

------

# 1. CNN이 필요한 이유

이미지는 단순한 숫자의 집합처럼 보이지만, 실제로는 **위치 정보와 패턴**이 매우 중요하다.
예를 들어, 같은 픽셀 값이라도 위치가 다르면 전혀 다른 의미를 가지게 된다.

일반 신경망은 이러한 공간 구조를 고려하지 못하기 때문에
이미지 인식에서는 한계가 있다.

반면 CNN은
👉 **이미지를 작은 영역 단위로 보면서 패턴을 학습**하기 때문에
훨씬 효과적으로 특징을 이해할 수 있다.

------

# 2. CNN 전체 구조 흐름

CNN은 다음과 같은 단계로 이루어진다.

```
Convolution → ReLU → Pooling → Flatten → Fully Connected → Output
```

이 흐름은 크게 두 부분으로 나눌 수 있다.

- 앞부분 → 이미지에서 특징을 추출하는 과정
- 뒷부분 → 추출된 특징을 이용해 결과를 판단하는 과정

------

# 3. 단계별 동작 원리

## ① Convolution (합성곱)

Convolution 단계에서는 이미지 위에 **필터(filter)**를 올려놓고,
이 필터를 한 칸씩 이동시키면서 이미지의 특징을 찾아낸다.

이 필터는 일종의 “탐지기” 역할을 하며,
특정 패턴이 존재하는지를 확인한다.

예를 들어

- 밝기 변화 → 엣지(경계선)
- 특정 모양 → 눈, 코, 귀 같은 구조

이 과정을 통해 만들어지는 결과를
👉 **Feature Map(특징 맵)**이라고 한다.

👉 한 줄 정리
→ “이미지를 훑으면서 의미 있는 특징을 찾아낸다”

------

## ② ReLU (Rectified Linear Unit)

Convolution을 통해 얻은 값에는
중요하지 않은 정보도 많이 포함되어 있다.

ReLU는 이 중에서
👉 **음수 값을 0으로 바꾸고, 양수 값만 남기는 역할**을 한다.

이 과정은 단순해 보이지만 매우 중요하다.

왜냐하면, 신경망이 단순한 선형 구조만 가지게 되면
복잡한 패턴을 학습할 수 없기 때문이다.

ReLU를 사용하면
👉 **비선형성을 추가하여 더 다양한 패턴을 학습할 수 있게 된다.**

👉 한 줄 정리
→ “중요한 정보만 남기고 불필요한 값은 제거한다”

------

## ③ Pooling

Pooling 단계에서는 Feature Map의 크기를 줄인다.

이미지의 모든 정보를 그대로 유지하면
계산량이 너무 커지고, 불필요한 정보도 많아지기 때문이다.

대표적으로 **Max Pooling**을 사용하며,
이는 특정 영역에서 가장 큰 값만 선택하는 방식이다.

이렇게 하면
👉 중요한 특징은 유지하면서 데이터 크기를 줄일 수 있다.

👉 한 줄 정리
→ “핵심 특징만 남기고 정보를 압축한다”

------

## ④ Flatten

Pooling까지 끝나면 데이터는 여전히 2차원 구조를 가지고 있다.

하지만 이후 단계에서는 일반 신경망을 사용해야 하기 때문에
이 데이터를 1차원으로 변환해야 한다.

Flatten은 이 과정을 수행하며,
👉 **2차원 데이터를 하나의 긴 벡터로 펼치는 역할**을 한다.

👉 한 줄 정리
→ “이미지 형태를 숫자 리스트로 바꾼다”

------

## ⑤ Fully Connected Layer

이 단계는 우리가 흔히 알고 있는 일반 신경망과 동일하다.

Flatten된 데이터를 입력으로 받아
각 특징을 종합적으로 고려하여 최종 판단을 내린다.

예를 들어

- 눈이 있다
- 귀가 있다
- 털 패턴이 있다

이러한 정보를 종합하여
👉 “이건 고양이다”라고 판단하게 된다.

👉 한 줄 정리
→ “추출된 특징을 종합해서 결론을 내린다”

------

## ⑥ Output Layer

마지막 단계에서는 최종 결과를 출력한다.

이때 주로 사용하는 것이 **Softmax 함수**이며,
각 클래스에 대한 확률을 계산해준다.

예를 들어

- 고양이: 0.9
- 강아지: 0.1

이처럼 확률 형태로 결과가 나오며
가장 높은 값을 가진 클래스가 최종 예측이 된다.

또한 학습 과정에서는 **Cross-Entropy**를 사용하여
예측값과 실제값의 차이를 계산하고,
이 오차를 줄이는 방향으로 모델이 학습된다.

👉 한 줄 정리
→ “각 결과의 확률을 계산하고 최종 답을 선택한다”

------

# 4. 전체 흐름 요약

CNN의 전체 과정은 다음과 같이 이해할 수 있다.

- Convolution → 이미지에서 특징을 찾는다
- ReLU → 중요한 정보만 남긴다
- Pooling → 정보를 압축한다
- Flatten → 데이터를 펼친다
- Fully Connected → 판단한다
- Output → 결과를 출력한다

------

# 5. 핵심 요약

CNN은 단순히 이미지를 입력받는 것이 아니라,
이미지를 여러 단계에 걸쳐 분석하면서
점점 더 중요한 특징만 남기는 구조를 가진다.

그리고 마지막 단계에서
이 특징들을 종합하여 최종 결과를 도출한다.

------
