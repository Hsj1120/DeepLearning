# 🧠 Softmax & Cross-Entropy 쉽게 이해하기

CNN의 마지막 단계에서는
👉 **“이게 무엇인지” 확률로 판단**한다

![4-33](../img\section4\4-23.png)

하지만 2개의 최종 뉴런이 서로 연결되어 있을 이유가 없는데 어떻게 상대 뉴런의 값을 알고 1을 만들 수 있을까??

이때 사용되는 개념이 바로

👉 **Softmax + Cross-Entropy**이다

------

# 1. Softmax란 무엇인가?

CNN의 마지막 출력층에서는
각 클래스에 대한 값이 나온다.

문제는

👉 이 값들은 그냥 숫자일 뿐
👉 **확률이 아니다**

------

예를 들어

```id="soft1"
dog = 2.3
cat = 1.1
```

이건 확률이 아니다.

------

그래서 사용하는 것이

👉 **Softmax 함수**

------

## ✔ Softmax의 역할

![4-33](../img\section4\4-24.png)

- 값을 0 ~ 1 사이로 변환
- 전체 합이 1이 되도록 조정

------

👉 결과

```id="soft2"
dog = 0.9
cat = 0.1
```

------

👉 한 줄 정리
→ “출력을 확률로 만들어준다”

------

## ✔ 왜 필요한가?

예를 들어

- dog = 80%
- cat = 45%

👉 이런 건 말이 안 된다

→ 확률 합이 100%가 아니기 때문

------

👉 Softmax를 쓰면
👉 항상 합이 1이 된다

------

# 2. Cross-Entropy란 무엇인가?

![4-25](../img\section4\4-25.png)

Softmax로 확률을 만들었으면

👉 이제 해야 할 것

👉 **“얼마나 틀렸는지 계산”**

------

그 역할을 하는 것이

👉 **Cross-Entropy (손실 함수)**이다

------

👉 한 줄 정리
→ “예측이 얼마나 틀렸는지 측정”

------

# 3. Cross-Entropy가 하는 일

예를 들어

![4-25](../img\section4\4-26.png)

👉 실제 정답 = dog

------

![4-27](../img\section4\4-27.png)

## ✔ 예측 1

```id="ce1"
dog = 0.9
cat = 0.1
```

👉 거의 정답 → 오차 작음

------

## ✔ 예측 2

```id="ce2"
dog = 0.6
cat = 0.4
```

👉 애매함 → 오차 중간

------

## ✔ 예측 3

```id="ce3"
dog = 0.1
cat = 0.9
```

👉 완전 틀림 → 오차 큼

------

👉 Cross-Entropy는
👉 이런 차이를 정확하게 반영한다

------

# 4. 왜 Cross-Entropy를 사용할까?

우리가 이전에 배운 방법은

👉 **MSE (Mean Squared Error)**

------

하지만 문제점이 있다

👉 “얼마나 틀렸는지”를 잘 반영하지 못함

------

## ✔ 핵심 차이

- MSE → 단순 거리 계산
- Cross-Entropy → 확률 기반 비교

------

👉 특히 중요한 점

👉 Cross-Entropy는

👉 **틀린 예측에 더 강하게 패널티를 준다**

------

👉 그래서

👉 학습이 훨씬 빠르고 정확해진다

------

# 5. Softmax + Cross-Entropy 관계

이 둘은 항상 같이 사용된다.

------

## ✔ 흐름

```id="softflow"
출력값 → Softmax → 확률
→ Cross-Entropy → 오차 계산
```

------

👉 의미

- Softmax → 확률 만들기
- Cross-Entropy → 틀린 정도 측정

------

👉 한 줄 정리
→ “확률 만들고 → 틀린 정도 계산”

------

# 6. 직관적으로 이해하기

이걸 사람 기준으로 보면

------

### ✔ Softmax

👉 “이게 강아지일 확률 90%야”

------

### ✔ Cross-Entropy

👉 “근데 실제 정답이 강아지니까
→ 이건 꽤 잘 맞춘 거야”

------

👉 반대로

👉 “고양이 90%”라고 하면
→ 엄청 크게 틀린 것

------

# 7. 언제 사용하는가?

👉 Cross-Entropy는

👉 **분류 문제에서 사용**

------

👉 MSE는

👉 **회귀 문제에서 사용**

------

# 8. 핵심 요약

- Softmax는 출력을 확률로 변환한다
- Cross-Entropy는 오차를 계산한다
- 둘은 항상 함께 사용된다
- 분류 문제에서 가장 효과적인 조합이다

------

# 🎯 한 줄 정리

👉 **“Softmax는 확률을 만들고, Cross-Entropy는 얼마나 틀렸는지 측정한다.”**

------

# 💡 이 글에서 가장 중요한 감각

👉 “Softmax = 확률화”
👉 “Cross-Entropy = 오차 계산”

------

# 🔥 CNN 전체 마지막 연결

이제 완전히 이어진다

```id="cnnfinal"
이미지
→ 특징 추출
→ 특징 압축
→ 벡터 변환
→ 분류 (Softmax)
→ 오차 계산 (Cross-Entropy)
→ 학습
```
