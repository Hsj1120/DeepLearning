# 1. 이번 강의에서 배우는 것

이전 강의에서 SOM 학습을 끝냈다.

이번 강의에서는 **학습된 SOM 결과를 시각화한다.**

------

# 2. 우리가 보고 싶은 것

SOM 결과는 2차원 grid로 표현된다.

여기서 우리가 가장 중요하게 볼 값은 **MID**이다.

MID는 **Mean Interneuron Distance**이다.

뜻은:

```text
한 neuron과 주변 neuron들 사이의 평균 거리
```

이다.

------

# 3. MID가 중요한 이유

MID가 크다는 것은:

```text
그 neuron이 주변 neuron들과 멀리 떨어져 있다
```

는 뜻이다.

즉:

```text
MID 큼
→ 주변과 다름
→ outlier일 가능성 높음
→ 잠재적 fraud 가능성 높음
```

이다.

------

# 4. 색깔로 MID를 표시한다

숫자로 MID를 하나하나 비교하지 않는다.

색깔로 표시한다.

강의에서는:

```text
MID가 클수록 흰색에 가까움
MID가 작을수록 어두운 색에 가까움
```

으로 표현한다.

👉 그래서 흰색에 가까운 neuron을 보면 outlier 후보라고 볼 수 있다.

------

# 5. 필요한 함수 import

SOM 시각화를 위해 `pylab`에서 몇 가지 함수를 가져온다.

```python
from pylab import bone, pcolor, colorbar, plot, show
```

각 함수의 역할은 대략 이렇다.

```text
bone      → 그래프 배경 스타일 설정
pcolor    → SOM grid를 색깔로 표시
colorbar  → 오른쪽 색상 범례 표시
plot      → marker 표시
show      → 그래프 출력
```

------

# 6. 빈 그래프 창 만들기

먼저 그래프 창을 만든다.

```python
bone()
```

이 코드는
SOM 지도를 그릴 기본 화면을 준비한다.

------

# 7. Distance Map 그리기

이제 SOM의 MID 값을 색깔로 표시한다.

```python
pcolor(som.distance_map().T)
```

여기서 핵심은:

```python
som.distance_map()
```

이다.

이 함수는 SOM의 각 neuron에 대한 MID 값을 matrix 형태로 반환한다.

------

# 8. 왜 `.T`를 붙일까?

```python
.T
```

는 transpose를 의미한다.

즉, 행과 열을 바꾸는 것이다.

```text
matrix 방향을 시각화에 맞게 돌려주는 작업
```

이라고 보면 된다.

------

# 9. Colorbar 추가

색깔이 어떤 값을 의미하는지 보기 위해
범례를 추가한다.

```python
colorbar()
```

그러면 오른쪽에 색깔 막대가 생긴다.

이 막대를 보면:

```text
어두운 색 = 낮은 MID
흰색 = 높은 MID
```

임을 확인할 수 있다.

------

# 10. 여기까지만 해도 outlier를 볼 수 있다

이 단계에서 이미 outlier를 찾을 수 있다.

흰색에 가까운 winning node가 있다면:

```text
주변 neuron들과 많이 다름
→ 일반 규칙에서 벗어남
→ fraud 가능성 있음
```

으로 해석할 수 있다.

------

# 11. 하지만 marker를 추가하면 더 좋다

우리는 단순히 outlier만 보고 싶은 것이 아니다.

그 outlier에 해당하는 고객이:

```text
카드 승인을 받은 고객인지
승인받지 못한 고객인지
```

도 보고 싶다.

왜냐하면 은행 입장에서는:

```text
사기 의심 + 승인받은 고객
```

이 더 중요하기 때문이다.

------

# 12. Marker 설정

두 종류의 marker를 만든다.

```python
markers = ['o', 's']
colors = ['r', 'g']
```

뜻은:

```text
'o' = 원
's' = 사각형

'r' = 빨간색
'g' = 초록색
```

강의에서는:

```text
빨간 원 = 승인 안 된 고객
초록 사각형 = 승인된 고객
```

으로 사용한다.

------

# 13. 모든 고객을 반복한다

이제 모든 고객을 하나씩 돌면서
각 고객의 winning node를 찾는다.

```python
for i, x in enumerate(X):
```

여기서:

```text
i = 고객의 index
x = 고객 한 명의 데이터 vector
```

이다.

예를 들어 첫 반복에서는:

```text
i = 0
x = 첫 번째 고객 데이터
```

가 된다.

------

# 14. 고객의 winning node 찾기

각 고객에 대해 winning node를 찾는다.

```python
w = som.winner(x)
```

뜻은:

```text
현재 고객 x와 가장 가까운 SOM neuron의 위치를 찾는다
```

이다.

즉, 이 고객이 SOM 지도에서 어느 칸에 매칭되는지 찾는 것이다.

------

# 15. marker를 어디에 찍을까?

winning node의 위치는:

```python
w[0], w[1]
```

이다.

하지만 이 좌표는 grid 칸의 왼쪽 아래 모서리 기준이다.

그래서 marker를 칸 중앙에 찍으려면:

```python
w[0] + 0.5
w[1] + 0.5
```

를 사용한다.

------

# 16. y[i]로 승인 여부 확인

각 고객이 승인되었는지 아닌지는 `y`에 들어 있다.

```text
y[i] = 0 → 승인 안 됨
y[i] = 1 → 승인됨
```

그래서 marker와 color를 이렇게 선택한다.

```python
markers[y[i]]
colors[y[i]]
```

즉:

```text
y[i] = 0 → markers[0] = 'o', colors[0] = 'r'
y[i] = 1 → markers[1] = 's', colors[1] = 'g'
```

------

# 17. marker 그리기

전체 코드는 이런 형태이다.

```python
plot(w[0] + 0.5,
     w[1] + 0.5,
     markers[y[i]],
     markeredgecolor = colors[y[i]],
     markerfacecolor = 'None',
     markersize = 10,
     markeredgewidth = 2)
```

뜻은:

```text
winning node 중앙에 marker를 찍는다
고객 승인 여부에 따라 marker 모양과 색을 다르게 한다
```

------

# 18. markerfacecolor를 None으로 하는 이유

```python
markerfacecolor = 'None'
```

은 marker 안쪽을 비워두겠다는 뜻이다.

왜 이렇게 할까?

하나의 winning node에
여러 고객이 매칭될 수 있기 때문이다.

안쪽을 채워버리면
marker가 겹쳤을 때 보기 어렵다.

그래서 테두리만 색칠한다.

------

# 19. 최종 출력

마지막으로 그래프를 보여준다.

```python
show()
```

그러면 SOM map이 출력된다.

이 map에서는 두 가지 정보를 동시에 볼 수 있다.

```text
배경 색깔 = MID 값
marker = 고객 승인 여부
```

------

# 20. 결과 해석

흰색에 가까운 칸은 MID가 높다.

즉:

```text
outlier neuron
→ fraud 가능성 높음
```

이다.

그 칸에 초록 사각형이 있다면:

```text
사기 의심 고객인데 카드 승인도 받음
```

이라는 뜻이다.

은행 입장에서는 이런 고객이 더 중요하다.

------

# 21. 이번 강의 전체 코드

```python
from pylab import bone, pcolor, colorbar, plot, show

bone()
pcolor(som.distance_map().T)
colorbar()

markers = ['o', 's']
colors = ['r', 'g']

for i, x in enumerate(X):
    w = som.winner(x)
    plot(w[0] + 0.5,
         w[1] + 0.5,
         markers[y[i]],
         markeredgecolor = colors[y[i]],
         markerfacecolor = 'None',
         markersize = 10,
         markeredgewidth = 2)

show()
```

# 22. 전체 흐름 정리

```text
SOM 학습 완료
→ 각 neuron의 MID 계산
→ MID를 색깔로 표시
→ 고객별 winning node 찾기
→ 승인 여부에 따라 marker 표시
→ 흰색 영역의 marker 확인
→ fraud 의심 고객 후보 파악
```
