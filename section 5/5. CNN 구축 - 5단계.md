**새로운 이미지 하나 넣어서 예측하는 단계 (Inference)**

------

# 📌 전체 흐름

```text
이미지 불러오기 → 전처리 → 배치 추가 → 예측 → 결과 해석
```

------

# 📌 1. 라이브러리 준비

```python
import numpy as np
from keras.preprocessing import image
```

👉 이미지 처리 + 배열 변환을 위해 필요

------

# 📌 2. 이미지 불러오기

```python
test_image = image.load_img(
    'dataset/single_prediction/cat_or_dog_1.jpg',
    target_size=(64, 64)
)
```

------

## ✔ 핵심 포인트

👉 **훈련할 때 사용한 이미지 크기와 동일해야 함**

- 우리는 `(64, 64)` 사용했음
- CNN 입력도 `(64,64,3)`

------

# 📌 3. 이미지 → 배열 변환

```python
test_image = image.img_to_array(test_image)
```

------

## ✔ 이유

👉 CNN은 이미지를 그대로 못 받음
👉 **숫자 배열 형태로 변환 필요**

------

# 📌 4. 배치 차원 추가 (🔥 중요)

```python
test_image = np.expand_dims(test_image, axis=0)
```

------

## ✔ 왜 필요?

👉 CNN은 이렇게 입력 받음

```text
(batch_size, height, width, channel)
```

------

## ✔ 지금 상황

👉 이미지 1장이라도

```text
(1, 64, 64, 3)
```

👉 이렇게 만들어줘야 함

------

# 📌 5. 예측 수행

```python
result = cnn.predict(test_image)
```

------

## ✔ 결과

```text
[[0.87]]
```

👉 의미:

👉 “개일 확률 87%”

------

# 📌 6. label 확인 (🔥 중요)

```python
training_set.class_indices
```

👉 예시 출력:

```text
{'cats': 0, 'dogs': 1}
```

------

# 📌 7. 결과 해석

```python
if result[0][0] == 1:
    prediction = 'dog'
else:
    prediction = 'cat'

print(prediction)
```

------

## ✔ 핵심

👉 result는 **배치 형태**

```text
[[0.87]]
```

👉 그래서

```python
result[0][0]
```

👉 이렇게 접근해야 함

------

import numpy as np

    'dataset/single_prediction/cat_or_dog_1.jpg',

)

# 배열 변환
test_image = image.img_to_array(test_image)

# 배치 차원 추가
test_image = np.expand_dims(test_image, axis=0)

# 예측

👉 이 단계에서 중요한 것 3가지
