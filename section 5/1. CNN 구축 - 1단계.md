# 📌 CNN(컨볼루션 신경망) 입문 정리

## 1. CNN이란?

컨볼루션 신경망(CNN)은
👉 이미지 데이터를 처리하기 위해 만들어진 딥러닝 모델이다.

기존의 인공신경망(ANN)은 숫자 형태의 데이터만 다루는 데 강했지만,
CNN은 이미지처럼 **공간 정보를 가진 데이터**도 처리할 수 있다.

쉽게 말하면
👉 **AI에게 ‘눈’을 달아주는 기술**이라고 보면 된다.

------

## 2. CNN의 특징

- 이미지, 영상, 3D 데이터 처리 가능
- 특징(feature)을 자동으로 추출
- 사람처럼 이미지를 “보고” 판단

------

## 3. 왜 Python을 사용할까?

CNN 구현은 보통 Python을 사용한다.

그 이유는:

- 딥러닝 라이브러리가 매우 강력함
  - TensorFlow (구글)
  - PyTorch (페이스북)
- 컴퓨터 비전 분야에서 사실상 표준

👉 R은 통계 분석에 강하지만
👉 **이미지 처리(컴퓨터 비전)는 Python이 압도적으로 유리**

------

## 4. 실습 문제: 고양이 vs 개 분류

이번 실습의 목표는 간단하다.

👉 **이미지를 보고 고양이인지 개인지 맞추는 모델 만들기**

### 📂 데이터 구성

- **훈련 데이터**
  - 고양이: 4,000장
  - 개: 4,000장
  - 총 8,000장
- **테스트 데이터**
  - 고양이: 1,000장
  - 개: 1,000장
  - 총 2,000장
- **추가 테스트용 이미지**
  - 단일 이미지로 예측 테스트

👉 다양한 품종과 형태의 이미지가 포함되어 있어
모델이 일반화된 특징을 학습하게 된다.

------

## 5. 왜 데이터셋이 따로 제공될까?

이미지 데이터는 용량이 매우 크다.

- 수백 MB 이상
- 프로젝트 다운로드 시 부담됨

그래서:
👉 **코드와 데이터셋을 분리해서 제공**

## 7. CNN 전체 구조 (핵심 ⭐)

CNN은 다음 4단계로 구성된다.

------

### 🔹 Part 1: 데이터 전처리

이미지 데이터를 모델이 사용할 수 있게 변환

- 훈련 데이터 전처리
- 테스트 데이터 전처리

👉 기존 숫자 데이터와 다르게
👉 **이미지 전처리가 필요**

------

### 🔹 Part 2: CNN 모델 구축

순서가 매우 중요 👇

1. **Convolution (합성곱)**
   - 특징 추출
2. **Pooling (풀링)**
   - 데이터 크기 축소 (Max Pooling)
3. **Convolution 추가**
   - 더 깊은 특징 학습
4. **Flatten (펼치기)**
   - 2D → 1D 변환
5. **Fully Connected Layer**
   - 최종 판단

------

### 🔹 Part 3: 모델 학습

- 모델 컴파일
- 훈련 데이터로 학습
- 테스트 데이터로 평가

------

### 🔹 Part 4: 예측

- 새로운 이미지 입력
- 고양이 vs 개 판별

------

## 8. 핵심 흐름 한 줄 요약

👉 이미지 입력
→ 특징 추출 (CNN)
→ 벡터 변환
→ 분류 (고양이 or 개)

------
