# 📌 CNN 모델 구축 (Part 2 핵심 정리)

👉 이 단계는
**데이터를 학습하는 단계가 아니라**
👉 **모델 구조를 설계하는 단계**

------

# 📌 1. CNN 초기화

```python
cnn = tf.keras.models.Sequential()
```

👉 의미:

- CNN을 “레이어들의 집합”으로 만드는 틀
- 레이어를 하나씩 쌓아가는 구조

------

## ✔ 핵심 개념

👉 Sequential = **순서대로 쌓는 모델**

```text
입력 → 레이어 → 레이어 → 출력
```

------

# 📌 2. Convolution Layer (Step 1 - 특징 추출)

```python
cnn.add(tf.keras.layers.Conv2D(
    filters=32,
    kernel_size=3,
    activation='relu',
    input_shape=(64, 64, 3)
))
```

------

## ✔ `cnn.add()`의 의미 (🔥 중요)

👉 `cnn.add()`는

👉 **CNN 모델에 레이어를 하나 추가하는 함수**

------

## 🧠 비유

👉 건물 쌓기

- `Sequential()` → 빈 건물
- `cnn.add()` → 층 추가

```python
cnn.add(Conv)   # 1층
cnn.add(Pool)   # 2층
cnn.add(Dense)  # 3층
```

👉 이렇게 위로 계속 쌓는다

------

## ✔ 자동 연결 (중요)

👉 `add()`를 하면

👉 이전 레이어 출력 → 다음 레이어 입력
👉 자동 연결됨

------

## ✔ 핵심 파라미터

- `filters=32`
  👉 특징 감지기 32개
- `kernel_size=3`
  👉 3x3 영역을 보면서 특징 추출
- `activation='relu'`
  👉 음수 제거 → 학습 효율 ↑
- `input_shape=(64,64,3)`
  👉 입력 이미지 크기 + RGB (컬러)

------

## ✔ 한 줄 이해

👉
“cnn.add()로 레이어를 쌓고, Conv2D로 이미지 특징을 추출한다”

------

# 📌 3. Pooling Layer (Step 2 - 크기 축소)

```python
cnn.add(tf.keras.layers.MaxPool2D(
    pool_size=2, // 2 * 2 사이즈로
    strides=2 // 2칸씩 겹치지 않게
))
```

------

## ✔ 역할

👉 중요한 정보만 남기고 크기 줄이기

------

## ✔ 왜 필요?

- 계산량 감소
- 과적합 방지
- 위치 변화에 강함

------

## ✔ 한 줄 이해

👉
“이미지를 압축하면서 핵심 정보만 남긴다”

------

# 📌 4. Convolution + Pooling 한 번 더

```python
cnn.add(tf.keras.layers.Conv2D(filters=32, kernel_size=3, activation='relu'))
cnn.add(tf.keras.layers.MaxPool2D(pool_size=2, strides=2))
```

------

## ✔ 왜 또 하냐? (🔥 핵심)

👉 **더 복잡한 특징을 학습하기 위해**

------

## ✔ 계층적 학습

- 1층: 선 (edge)
- 2층: 눈, 귀
- 3층: 얼굴

👉 점점 더 “의미 있는 특징”으로 발전

------

## ✔ 핵심 포인트

👉 두 번째 Conv는
👉 **원본 이미지가 아니라 feature map을 입력으로 받는다**

------

# 📌 5. Flatten (Step 3 - 펼치기)

```python
cnn.add(tf.keras.layers.Flatten())
```

------

## ✔ 역할

👉 2D 데이터를 1D 벡터로 변환

------

## ✔ 왜 필요?

👉 Dense Layer는 1차원만 입력 가능

------

## ✔ 한 줄 이해

👉
“이미지를 숫자 리스트로 펼친다”

------

# 📌 6. Fully Connected Layer (Step 4 - 판단)

```python
cnn.add(tf.keras.layers.Dense(units=128, activation='relu'))
```

------

## ✔ 역할

👉 최종 판단을 위한 “두뇌”

------

## ✔ 왜 128?

👉 문제 복잡도가 높아서 더 많은 뉴런 필요

------

# 📌 7. Output Layer

```python
cnn.add(tf.keras.layers.Dense(units=1, activation='sigmoid'))
```

------

## ✔ 핵심

- `units=1`
  👉 이진 분류 (고양이 vs 개)
- `sigmoid`
  👉 0 ~ 1 확률 출력

------

# 📌 전체 구조 한눈에

```text
이미지 입력
→ Conv
→ Pool
→ Conv
→ Pool
→ Flatten
→ Dense
→ Output
```

------

# 🔥 핵심 요약

👉 CNN은

- 앞부분: 특징 추출 (Conv + Pool)
- 뒷부분: 판단 (Dense)

------

# 💡 진짜 중요한 포인트

👉 `input_shape`는 **첫 레이어만 설정**

👉 이후 레이어는 자동 연결됨 (`cnn.add()` 덕분)

------

# 💡 한 줄 핵심

👉
“CNN은 특징을 추출하고 → 그 특징을 기반으로 판단하는 구조”

------

# 💡 면접용 한 문장

👉
“CNN은 Convolution과 Pooling을 통해 특징을 추출하고, Fully Connected Layer에서 최종 분류를 수행합니다.”

------

# 🚀 추가 이해 포인트 (한 단계 위)

👉 Sequential = 구조
👉 add() = 레이어 추가
👉 Conv = 특징 추출
👉 Pool = 압축
👉 Dense = 판단
